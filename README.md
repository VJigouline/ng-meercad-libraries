## Development notes

Use local linking without exporting to NPM repository during development  
```powershell
# In library
cd dist/my-components
npm link
# In application
npm link my-components
# Don't forget to unlink when finished and converted into NPM module
npm unlink my-components
npm unlink -g my-components
```
